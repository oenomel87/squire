## 문서 구성

1. [01. 동작 환경 구성](./01-environment.md)
2. [02. 서버/웹 구동 방법](./02-run-server-web.md)
3. [03. CLI 커맨드 목록](./03-cli-commands.md)

## 권장 읽기 순서

1. `01-environment.md`로 환경을 준비합니다.
2. `02-run-server-web.md`로 백엔드/프론트를 실행합니다.
3. `03-cli-commands.md`로 실무 커맨드를 확인합니다.
4. 다른 프로젝트에서 재사용하려면 `04-cli-global-setup.md`를 진행합니다.
5. Codex 또는 Claude Code 연동 자동화를 원하면 `05-skill-guide.md`를 사용합니다.
